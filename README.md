## Project Overview
The Customer Order Analytics Dashboard is a data analytics project developed using Power BI to analyze customer behavior, order trends, and sales performance. This dashboard transforms raw customer and order data into meaningful insights that support data-driven decision-making for business stakeholders.

## Objectives
* Analyze customer purchasing patterns
* Track total orders, revenue, and order status
* Identify top customers and high-value orders
* Monitor order trends over time
* Support business decisions using visual insights

## Dataset Description
1) Customers Table
* Customer ID
* Customer Name
* Contact Details
* Location

 2) Orders Table
* Order ID
* Customer ID
* Order Date
* Order Status
* Total Amount

3) Order Items Table
* Order Item ID
* Order ID
* Product Name
* Quantity
* Price

## Key Dashboard Features
* Total Customers & Orders Overview
* Sales & Revenue Analysis
* Order Status Breakdown (Completed, Cancelled, Pending)
* Top Customers by Revenue
* Monthly / Yearly Order Trends
* Interactive Filters & Slicers
